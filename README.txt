Shoouji Tsyrdean

A simple interface for querying the CC-CEDICT Chinese-English dictionary.

Displays pronunciation in Gwoyeu Romatzyh and accepts search terms in GR.

STATIC CLASSES

Cedict holds a Properties file of dictionary entries

Conversion holds HashMaps to convert between GR and pinyin syllables

BUILDING

javac -encoding "UTF-8" CLI.java
javac -encoding "UTF-8" GUI.java
